# Reglas del proyecto

## Git - Commit y Push
**IMPORTANTE**: NUNCA hacer `git commit` o `git push` automáticamente. SIEMPRE preguntar al usuario antes de hacer commit y push usando la herramienta AskUserQuestion con opciones como:
- "Sí, hacer commit y push"
- "Solo commit (sin push)"
- "No, todavía no"

Esta regla es obligatoria y no puede ser ignorada.
